# Repositório do Projeto — Mi Patisserie (Doceria e Cafeteria)

Este repositório foi criado com o objetivo de centralizar e organizar todo o desenvolvimento do projeto **Mi Patisserie – Doceria e Cafeteria**, desenvolvido para o **Projeto Integrador (Inter)**.

A proposta é que este espaço no GitHub seja utilizado por **todos os integrantes do grupo** para acompanhar, registrar e contribuir com a evolução do projeto ao longo do semestre.

---

## Objetivo do repositório

O repositório servirá como um **ambiente de colaboração**, permitindo que todos os membros do grupo possam:

- Acompanhar o andamento do projeto  
- Visualizar alterações realizadas no sistema  
- Compartilhar códigos, documentos e arquivos relacionados  
- Manter uma organização clara das etapas do desenvolvimento  
- Registrar melhorias e atualizações feitas no projeto  

---

##  Como o repositório será utilizado

Durante o desenvolvimento do projeto, este repositório será utilizado para armazenar:

- Código do sistema (site ou aplicação)  
- Protótipos de interface (ex.: Figma)  
- Documentação do projeto  
- Modelagem de banco de dados  
- Arquivos de planejamento  
- Atualizações e versões do sistema  

Cada integrante poderá **contribuir com melhorias, ajustes e novas funcionalidades**, garantindo que todo o grupo consiga acompanhar as modificações feitas no projeto.
